Educational Outcomes Markup

Schema

  • New GoogleAPI type CollegeOrUniversityWithOutcomes that extends the schema.org type CollegeOrUniversity
  • with 3 new facet properties: fieldOfStudyFacet, companyFacet, and functionFacet, of the new type EducationalFacet
  • and 3 new outcomes properties: fieldOfStudyOutcomes, companyOutcomes, and functionOutcomes, of the new type EducationalOutcome

Summary

    <script type="application/ld+json">
    {
      "@context": "https://schema.googleapis.com",
      "@type": "CollegeOrUniversityWithOutcomes",  // New type extending "CollegeOrUniversity"

      // College info: name, address, website, any other global identifiers.

      // Facets to restrict the cohort to a subset of graduates from this college.
      // Three optional properties:  fieldOfStudyFacet, companyFacet, and functionFacet.
      // Each of type EducationalFacet, with name, identifier, optionally sameAs, and
      // optionally an image url for its logo.
      //
      //   - These facets may all be missing, for a college page (example)
      //   - or any of them can be present for one of the deeper slices (example)
      //   - or they can all be present together for a deep combination (example)

      // Number of students from this college matching facets (or all if no facets present).
      // Property numStudents

      // Outcomes of students from this college matching facets (or all if no facets present).
      // Three properties:  fieldOfStudyOutcomes, companyOutcomes, and functionOutcomes.
      // Each is an array of type EducationalOutcome, with all the same fields as
      // EducationalFacet, plus numStudents for that outcome.
    }
    </script>
